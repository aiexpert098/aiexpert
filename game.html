How to Play

  • **Objective:** Find the three-digit code to unlock the final door.
  • **Desktop Controls:** Use the mouse to **drag and orbit** the camera. **Hover** over objects to see a hint (bottom center). **Click** to interact.
  • **Mobile Controls:** **Tap** an object to focus the camera. **Tap again** to interact. Use a single finger to drag and orbit.
  • Solve the puzzles in the room to find the three digits of the final code (7, 4, 1).
